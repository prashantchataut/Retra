# Paperclip Create Agent

This skill requires a configured Paperclip organization, API base URL, credentials, reporting line, and explicit agent-creation task. None are present in Retra. Therefore no Paperclip API calls or agent creation are performed. The skill is retained for future use and its preconditions are documented rather than bypassed.
