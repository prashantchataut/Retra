# Mobile Android Design

Use Material 3, Compose state hoisting, dynamic color as an option, semantic content descriptions, 48dp touch targets, lazy containers, adaptive phone/tablet/foldable layouts, predictable back/deep-link behavior, and lifecycle-safe coroutines. Use bottom navigation on compact widths, navigation rail on medium widths, and multi-pane composition where expanded. Avoid unstable lambdas, unbounded recomposition, theme leaks, and gesture conflicts.
