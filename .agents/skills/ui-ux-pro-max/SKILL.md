# UI/UX Pro Max — Retra Snapshot

Apply to visual structure, interaction, typography, color, responsive behavior, animation, and accessibility. Priorities: accessibility; touch/interaction; performance; style; responsive layout; typography/color; animation; feedback; navigation; data visualization.

Workflow used for Retra:
1. Detect stack: Jetpack Compose / Material 3.
2. Generate a coherent design system before adding screens.
3. Apply design dials: variance 7, motion 5, density 6.
4. Use semantic tokens, 4/8dp spacing, 48dp Android touch targets, adaptive navigation, accessible contrast, reduced motion, and purposeful 150–300ms motion.
5. Do not fabricate database search results when the bundled search database is unavailable; use the canonical default rules and disclose the fallback.
6. Before delivery, review the app checklist in `references/pro-rules.md`.
