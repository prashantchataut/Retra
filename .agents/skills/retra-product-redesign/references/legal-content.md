# Legal Content Discovery

## Allowed Product Flows

- Import a ROM backup supplied by the user.
- Scan user-selected local folders.
- Apply IPS, UPS, or BPS patches to a compatible user-supplied base ROM.
- Open an official creator or project page.
- Download a homebrew/public release when redistribution permission is explicit and the exact file is checksum-pinned.
- Import a trusted catalog that passes Retra's validator.

## Prohibited Product Flows

- Bundling commercial Pokémon or Nintendo ROMs.
- Scraping or deep-linking unauthorized ROM-hosting sites for one-tap play.
- Circumventing source-page access controls.
- Treating a title, popularity, or fan-project status as proof of distribution rights.
- Downloading an unverified file and labeling it trusted.

## Verification Copy

Prefer concrete language such as:

- “Official creator page”
- “Published SHA-256 verified”
- “Permission to redistribute confirmed”
- “Bring a compatible base game you own”

Avoid vague claims such as “safe ROM,” “official game,” or “100% legal.”
