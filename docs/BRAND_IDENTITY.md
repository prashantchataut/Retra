# Retra Brand Identity

## Mark

The Retra mark is an original rounded cartridge-like silhouette containing an abstract **R**, a prism trail, and a small memory sparkle. It is intentionally not shaped like a protected Nintendo console, cartridge, Poké Ball, or commercial game icon.

Source assets:

- `branding/retra-logo.svg` — canonical square mark.
- `branding/retra-wordmark.svg` — presentation wordmark.
- `app/src/main/res/drawable-nodpi/retra_logo.png` — high-resolution in-app image.
- `app/src/main/res/drawable/ic_retra_foreground.xml` — adaptive and monochrome launcher foreground.
- `app/src/main/res/mipmap-*` — legacy launcher fallbacks.

The logo is applied to the adaptive launcher icon, Android 12+ splash screen, onboarding, navigation rail, top app bar, home hero, and profile surface.

## Product expression

- **Narrative anchor:** personal retro memories, not a generic file browser.
- **Visual variance:** 7/10 — asymmetric hero surfaces and bold artwork, but calm settings and diagnostic screens.
- **Motion intensity:** 5/10 — short crossfades and content transitions; no constant ambient motion.
- **Density:** 6/10 — game library and settings remain information-rich without shrinking touch targets.
- **Glass:** limited to major navigation and hero layers; technical text remains opaque.
- **Accessibility:** scalable typography, non-color status labels, reduced motion/transparency, high contrast, controller navigation, and large targets.

## Color roles

- Void Black `#07080D` — OLED/background anchor.
- Midnight Navy `#101424` — opaque technical surfaces.
- Retra Indigo `#7567FF` — primary interaction and brand signal.
- Prism Cyan `#45D9FF` — active/connected states.
- Memory Violet `#B38BFF` — emotional highlight.
- Save Mint `#59E1AF` — save integrity/success.
- Adventure Gold `#FFC766` — achievements and memory sparkle.
